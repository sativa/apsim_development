Soil nitrogen balance

Checking input values.
There are several places in the parameterisation where understanding of the nature of the parameter may vary.  Make sure you check the following with the documentation or the module developer.
  1. pH measurement method and model requirement
  2. Organic Carbon measurement method and model requirement
  3. Bulk Density and meaning for swelling soils
  4. Ammonium measurement technique
  5. Microbial Biomass Pools initialisation in conjunction with previous cropping history.
  6. Soil Temperature Descriptors for the site in question
