APSIM User Interface

 

Introduction

From version 4, APSIM comes with a new user interface that lets users configure simulations via a drag and drop centric interface. This interface, unlike APSFront, provides complete access to all APSIM parameters.

 

Getting started

 

When first started, the welcome screen is displayed asking you if you would like to create a new simulation or open an existing one.  A list of recently opened simulations may also be displayed.

 

To get started click 'Create new simulation' button and select a simulation that is closest to the type of simulation you want to build.

 

 

 

Once loaded, a simulation is displayed in tree view form.

Toolboxes

Creating simulations from scratch, or modifying existing ones involves dragging modules from one or more toolboxes. By default, the interface comes with a standard toolbox that can be accessed by choosing the 'Toolboxes' menu item and clicking on the Open toolbox drop down button.  Then from the drop down list of available toolboxes choose 'Standard'.

These modules can be dragged onto your simulation.

 

You can also create your own toolbox by by choosing 'Toolboxes' menu item and clicking the 'Manage toolboxes' button. The 'APSIM Options' popup window appears which allows you to add or remove toolboxes from the simulation.

 

 

Choose 'New ...' and when the 'Save As' window appears give the toolbox an appropriate name and save the toolbox file in a location of your choice.  When the toolbox has been successfully saved choose the 'Ok'  button and click the 'Open Toolbox' toolbar icon on the main window and your select your blank toolbox.

 

To add items to your toolbox simply drag and drop modules from your simulation onto your toolbox area at the bottom of the screen.  This is a great way to reuse pieces of simulation, or complete simulations, in different projects.

 

 

Running APSIM

Running APSIM is as simple as clicking on the Run button on the toolbar. The resulting output file and summary files can be viewed by clicking on outputfile and summaryfile in the simulation tree.
