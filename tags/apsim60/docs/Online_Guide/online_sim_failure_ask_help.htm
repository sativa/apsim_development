Simulation Failure

Asking the Help Desk

The level of assistance possible by the Help Desk will depend on the level of information you are able to supply within your request for assistance.  To assist in this, APSIM users are provided with a simple diagnostic tool in the form of a short batch file.  Use of this tool and the appropriate information will greatly assist the Help Desk in resolving each issue.

  1. Locate the diagnostic batch file in the apsuite/diagnost directory in your APSuite installation.

  2. Run the batch file from your operating system interface (eg double click in windows explorer)

  3. Follow all the instructions.  You will be asked to fill in a problem report.   The information you have recorded during your resolution effort will prove invaluable.  Answer all questions and make sure you report the appropriate APSuite Release Verison number in the appropriate place.  You will be notified regarding the file name of the file containing the information gleaned by the diagnostic batch file.

  4. Mail the text file created by the procedure to Help Desk. If the summary file from the simulation is not excessively long it would be advantageous to attach this file as well.

The APSIM Help Desk will try to find a resolution to the failure from the given information.  If necessary, the Help Desk operator may contact you for more information.  They may even ask for copies of input data for use in replicating the error.  Once the error has been identified and a work-around or resolution is available you will be notified.
