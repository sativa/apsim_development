User manual for APSoil3

 
 

Introduction
 

APSoil3 enables the user to store and present 'paddock level' soil resource information in forms suitable for presentation and use in simulation modelling. The program enables the inputting of field data and the calculation of available soil water and nitrogen based on known soil characteristics, including bulk density and plant available water capacity. The user has the choice of using soil characteristics developed by researchers for a wide range of soils of Australia's northern cropping region or may elect to input and base results on their own characteristics. APSoil3 provides the capability to develop a soil database for particular farms and paddocks which may be used to compare resource usage over time.

Whilst APSoil3 provides a means of presenting data in a form useful in presentation and discussion, it should be noted that 'best practice' in soil monitoring underlies the use of this program. It is recommended that this program be used in conjunction with the manual Soil matters-monitoring soil water and nutrients in dryland farming (Dalgliesh and Foale 1998) which is available in written form or at www.farmscape.tag.csiro.au.


Starting Out
 

The APSoil3 opening screen displays seven options showing the various
modules of APSoil down the left-hand side under the heading APSoil3:

    • Soil Characterisation
    • Samples
    • Results
    • Graph
    • Help
    • About
    • Exit APSoil

Overview

The Soil Characterisation module allows you to select a soil type, from the APSRU database, that best represents your sampling location or to create new soil characteristics based on your own data. It is also possible to create sampling profiles to suit your particular sampling needs.

The Samples module is used to input the data you collect from the field. The sample data is stored according to the name of the farmer, the farm, the paddock and the date that sampling took place.

The Results module presents the field data as available water and nitrogen based on the selected soil characteristics and depth layers.

The Graph module presents the available water and nitrogen in two separate graphs, one for Soil Water, and one for Soil Nitrate. You can edit the graphs to suit your requirements and/or print them for a hard copy.

Help provides links to the FARMSCAPE web site, which has current updates and user manuals, and contact details for CSIRO/APSRU. It also provides a link to this user manual.

About provides information on the development of the APSoil program

The Exit APSoil option will return you to your Windows desktop.


 

Using the APSoil Modules



Soil Characterisation

The Soil Characterisation Module allows you to select a soil type that best represents your soil from the APSRU database or create your own soil characteristics.

Click on the Characterisation option on the left hand side of the main screen

…You will see… a Soil Characterisation Data form. The toolbar has buttons for adding and deleting folders. A drop down box will automatically have APSRU highlighted and the tree will show folders for Characterisation Data and Profile Data.
 
 
 
 
 

The APSRU soil characterisation database is the default soil characterisation setting. You are able to examine entries in the APSRU database but cannot modify data.
 
 

To Peruse the APSRU Database

  1. Select Characterisation Data
  • Use the tree structure to drill down through Region/Area to /Soil
  • View the selected soil related characteristics
  • Move down to /crop
  • View soil and crop related characteristics for the soil and crop selected


 
 
 
 
 
 

To develop your own Soil Characteristics

  1. Set your profile intervals ie. the depth intervals used when taking the soil samples.
  • From the drop down box, select your name
  • Double-click on the Profile Data folder
  • Select the Default folder (create a new folder if you have already used the default, using the Add an item button on the toolbar)
  • Enter your profile details

  1. Enter your soil characterisation data:
  • Highlight the Characterisation Data folder
  • Click on the ‘add an item’ button in the menu bar 
  • Give the new folder a name for the region
  • With the ‘region’ folder still highlighted, click on the ‘add an item’ button again
  • Give the new folder the name of a significant town or district in the region
  • With the ‘local town/district ’ folder still highlighted, click on the ‘add an item’ button again

  • Give the new folder the name of  your soil type
  • Select the ‘soil type' folder
  • Enter data for GPS, BD(g/cc), DUL(%vol), SAT(%vol), OC (%), EC (%), and pH (Leave columns empty where no data available).


 

  • With the ‘soil type’ folder highlighted, click on the ‘add an item’ button again
  • Give the new folder the name of the crop you are growing
  • Select the ‘crop’ folder
  • Enter data for LL (%vol), PAWC will be calculated automatically


You can use this process to create any number of profiles and soil characterisation data folders.
 
 
 
 

Samples
 
 

The Samples module is used to input the field data that you collect. The sample data is stored according to the name of the farmer, the farm, the paddock and the date sampling took place.

Click on the Samples option

…You will see… a Sample Data form. The toolbar has buttons for adding and deleting folders. The tree display will show a single folder for Sample Data.

2. To enter data:

  • Click on the Sample Data folder
  • Click on the ‘add an item’ button in the menu bar
  • Select the New folder
  • Change the folder name to the farmer’s name
  • Enter the details on the Farmer Details form

  • After entering the farmer details, select the Farmer folder again
  • Click on the ‘add an item’ button in the menu bar
  • Give the new folder the name of the farm
  • With the Farm folder highlighted, click on the ‘add an item’ button again
  • Give the new folder the name of the paddock
  • Enter the paddock details (on the tree, not in the right hand box)

  • With the Paddock folder highlighted, click on the ‘add an item’ button
  • Label the new folder with the date of sampling
  • Enter the sample data.

The number of sampling replicates may be changed to suit the circumstances (on the right hand display table). Results are meaned where two or more replicates are used.

Where the user has developed more than one sampling profile, this may be changed using the drop-down box Profile. If intervals used during field sampling do not match the depth intervals available under Profile then a new profile will need to be created under Characterisation-Profile Data.
 
 


 
 
 

Results

The Results module presents the water and nitrogen measured in the field as available water and nitrogen, as determined using the soil characteristics selected. The top half of the screen displays the selected soil characteristic, the bottom half, the calculated available water and nitrogen. Available water and nitrogen will change as different soil characteristics are selected. Unless crop lower limit information is an available characteristic, available water and nitrogen will not be displayed and the bottom half of the screen will remain blank.

Click on the Results option

…You will see… a results form. The toolbar has buttons for printing the results and presenting them in APSIM compatible form. The screen is split horizontally into two with a tree and display table in each.

The top tree enables selection of the appropriate soil characteristic, drilling down through database/region/town and /crop. Both the APSRU and users databases may be accessed.

The bottom tree enables selection of the field data for calculation. Default displayed data is whatever was selected last (during the current APSoil session) in the Samples module, thus enabling data to be inputted, calculations undertaken and graphing done without having to re-select field data.
 
 
 

Data is presented as:

  • water available in the profile for the crop selected (in depth of water in mm-may be thought of the same way as rainfall)
  • nitrogen available in the profile (in kg/ha)

 

What if the sampling intervals that I used during sampling are different to those in the soil characteristic database?

APSoil is designed to interpolate data where samples have been taken at different intervals to those used in the characterisation database eg. 0-10, 10-60, 60 to 90cm compared to 0-15, 15-30, 30-60, 60-90 cm., recalculating the various soil parameters (BD, DUL and CLL) at the actual depths of sampling. This information is then used in both tabular and graphical presentation.

To Generate APSIM Files

Clicking on the Generate APSIM Files button on the Results toolbar allows data in APSIM compatible format to be generated from the data currently on the screen. Data available for transfer to APSIM files will depend on what soil information is included in the APSoil database but can include DUL, SAT, CLL, current soil water, bulk density, organic carbon, pH and current nitrate N. At present APSIM compatible data has to be transferred manually to the actual APSIM soil files, using cut and paste. To ensure that all data being used for simulation is presented using the same sampling intervals, sampling data (current soil water and nitrate) is re-calculated to the standard layer intervals used in the Soil Characterisation database (generally 0-15, 15-30, 30-60, 60-90, 90-120, 120-150, 150-180). In the case where overall sampling depth is shallower than the soil characterisation information, sampling data is only presented to the depth available. 
 

Graphs

This module allows presentation of available water and nitrogen in graphical form.

Click on the Graphs option

…You will see… a graphs form. The toolbar has buttons for printing and editing the graphs. The screen will show three tree displays (on the left) and two graphs:
 


The top tree display allows selection of the appropriate soil characteristic drilling down through database/region/town and /crop.
The middle tree enables selection of the paddock to be graphed.

The bottom tree shows all dates on which the selected paddock has been sampled and allows selection of one or more of these dates for presentation.

The soil water graph shows the drained upper limit and crop lower water limit and the actual water content at the time of sampling.

The soil nitrate graph shows the available nitrate in the soil sample.
Total available water and nitrogen are shown at the bottom of each graph.

NB: Nitrate N is presented in kg/ha for each layer and where wide sampling intervals are used (for example 60 or 90cm) may result in the appearance of N bulges in some depth layers. Although it is not possible to know the distribution of nitrate in more detail (unless analyses are undertaken using shorter depth intervals eg 15 or 30cm), it is reasonable to assume that it is distributed more evenly than this type of presentation would appear to show. Care should be taken when interpreting these graphs to ensure that users understand that what appear to be large concentrations of nitrate may only be caused by the layer thickness being used.

You can compare the results of two samples from the same paddock taken on different dates by selecting more than one date in the bottom tree display. This will display the results from both samples on each graph.

Comments may be entered in the box below the graphs before printing, however, these comments will not be saved when you move on to another module or close APSoil.
