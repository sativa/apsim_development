Soil nitrogen balance

Crop nitrogen requirements.
If crop Nitrogen data is scarce or incomplete, it is still possible to calculate estimates of the N requirement for an observed crop yield using the simple equation

Crop N = Grain N + Straw N + Root N

Each component can be taken from measurements or calculated using different methods such as suggested below.

Grain N can be calculated from grain N concentration and Yield
e.g. if grain N concentration is given as a percentage the formula is Grain N = GrainN%/100 * yield and will have units the same as yield.

Straw N can be calculated from the amount of non-grain above-ground biomass and an assumed C:N ratio.
e.g. Straw N = Straw / C:N ratio * 0.4 (where 0.40 is the amount of carbon in carbohydrate).
If the amount of straw in unknown an estimate can be calculated from a normal harvest index (HI).
e.g. Straw = Yield/HI - Yield

Root N can be estimated from crop size and an assumed C:N ration
e.g. Root N = Root weight / C:N ratio * 0.4 (where 0.40 is the amount of carbon in carbohydrate).
where Root Weight = Total Above Ground Biomass * Root:Shoot Ratio
The crop module developer may be able to assist with values for Root:Shoot Ratio.
