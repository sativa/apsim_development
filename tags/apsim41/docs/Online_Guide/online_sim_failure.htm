Simulation Failure

A failed simulation is simply a simulation that has not run to completion.   Failure to complete may be due to defects in any of the following areas:

  • User input data - e.g. missing or eroneous data

  • Model logic or design - e.g. bad logic in module communications or calculations

  • Model or Simulation Infrastructure software implementation - e.g. A programming error

In most cases a simulation failure can be resolved by following a methodical process of error tracing.
