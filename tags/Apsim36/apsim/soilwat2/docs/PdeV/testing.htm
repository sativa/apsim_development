Test procedures

There are 3 tests included here, and some other miscellaneous files that may be useful:

  • For posterity, John Dimes' soilwat1 runs, from which all the following are derived.
  • A simple test run to ensure each submodel runs without crashing. Results

  • Starting with a full profile, rain on it to initialise 1st stage drying. Watch this over the long term to see second stage drying, until profile is depleted. Results .

  • With the same artificial climate, rain small amounts during second stage to watch it switch back to 1st stage. Results.

  • Compare using a real climate with some measured evaporation from a lysimeter. Results.
