APSIM User Interface

 

Introduction

From version 4, APSIM comes with a new user interface that lets users configure simulations via a drag and drop centric interface. This interface, unlike APSFront, provides complete access to all APSIM parameters.

 

Getting started

 

When first started, the welcome screen is displayed asking you if you would like to create a new simulation or open an existing one.  A list of recently opened simulations may also be displayed.

 

To get started click 'Create new simulation' button and select a simulation that is closest to the type of simulation you want to build.

 

 

 

Once loaded, a simulation is displayed in tree view form.

Toolboxes

Creating simulations from scratch, or modifying existing ones involves dragging modules from one or more toolboxes. By default, the interface comes with a standard toolbox that can be accessed by clicking on Standard button at the bottom of the screen. (NB. When the user interface is in "Full-Screen" mode, this button may be obscured by the Windows Start menu)

These modules can be dragged onto your paddock.

 

You can also create your own toolbox by clicking on Manage at the bottom of the screen.

 

 

Choose 'New ...' and when the 'Save As' window appears give the toolbox an appropriate name and save the toolbox file in a location of your choice.  When the toolbox has been successfully saved choose the 'Ok'  button. Your toolbox name will appear at the bottom of the window. Clicking it will open it and you will be able to drag items from your simulation into your newly created toolbox. This is a great way to reuse pieces of simulation, or complete simulations, in different projects.

 

 

 

Running APSIM

Running APSIM is as simple as clicking on the Run button on the toolbar. The resulting output file and summary files can be viewed by clicking on outputfile and summaryfile in the simulation tree.
