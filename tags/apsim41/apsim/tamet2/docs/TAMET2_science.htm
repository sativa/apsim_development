The APSIM TAMET2 module

 

APSIM TAMET2 module is adapted from TAMET2, which was a program, developed by B. H. Wall of the CSIRO Div. of Tropical Crops and Pastures. It was designed to test for errors in the met data collected by the Division's network of automatic met stations and from the Bureau of Meteorology.


This implementation checks for the following possible errors –

  • High or very high Evaporation with rain
  • Discontinuity in minT
  • Discontinuity in maxT
  • Discontinuity in evap
  • Rainfall outside limits
  • Evaporation outside limits
  • MaxT outside limits
  • MinT outside limits
  • Wind run outside limits
  • Radiation very low
  • Radiation outside limits
  • Radiation values well below clear sky value
  • MaxT < or = MinT
  • MaxT is close to MinT and not overcast

Limits and discontinuities are specified in the parameter file for the site being checked.

Note a GUI version of the same code is available at http://www.apsim.info/apsim/Tools/tamet.exe , and documentation on its use at http://www.apsim.info/apsim/Tools/Using%20TAMET2.htm .
