Crop growth & development

Step 4 - Weighing Up the Information

After a thorough examination of model and observed crop behaviour you should be in a suitable position to make some judgements regarding the applicability of the model and its configuration.

The following points should be kept in mind

  1. The model will not predict everything perfectly.

  2. Not all factors affecting growth are within the scope of the model (e.g. disease)

  3. Garbage-In-Garbage-Out - A simulation will only be as good as the input data.

  4. Your observed data will always have error associated with it.

  5. It is not always desirable for a model to simulate questionable data.

  6. Your input data description is always going to be incomplete.

  7. Indescriminate adjustment of parameters is undesirable.
