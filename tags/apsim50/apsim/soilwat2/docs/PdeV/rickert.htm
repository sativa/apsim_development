Ken Rickert's soil evaporation model.

This model may only be of interest to those wishing to run APSGRASP with a similar water balance to that used by McKeon.

Soil evaporation (ES) from layers 1 and 2 depends on the relative ability of either layer to meet the evaporative demand. The top layer can dry to airdry, whereas the second layer dries to ll15.

Each layer has a supply ratio (SI) that determines the amount of soil water the soil can provide for evaporation.

    ESk = SIk * EOS
where EOS is calculated elsewhere.

The supply ratio for the top layer is given by

    SI1 = 0.285 * AWR1 / (1.0 - 0.715 * AWR1 )
where
    AWRi is the available water ratio calculated with air dry values (AWR = (SW - AD)/(FC - AD)), and
    0.285 is a magic number determined experimentally (?).
For the second layer, SI2 is calculated by the difference between the supply ratio for both layers 1 and 2, and SI1.

    SI12 = 0.117 * AWR12 / (1.0 - 0.883 * AWR12 )
where 
 
        SW1 + SW2 - AD1 - AP2
AWR12 = ---------------------
        FC1 + FC2 - AD1 - WP2
so
    SI2 = SI12 - SI1

The total daily soil evaporation is limited by a parameter max_evap, ie

    ES1 + ES2 <= max_evap

A warning message will appear if total soil evaporation exceeds available soil water.

Note that McKeon's implementation allows the second layer to evaporate to the plant's wilting point, unlike the top layer which can dry to airdry. Is this a property of the evaporation process, or a reflection that he doesn't have airdry for the lower layers in his model?
