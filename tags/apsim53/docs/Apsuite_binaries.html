APSIM Binaries

 

Windows

APSIM comprises several applications for different tasks.

  • APSIM.EXE, a console application, is the core systems model; its single argument is one .sim file. Summary information is written to standard output, errors to standard error.
  • APSRun.exe converts any .apsim, .con, .run file to individual .sim files, and runs apsim.exe on each. Standard output is collected and written to a summary file, and also displayed in a text window. When run from a command line, the optional "/auto" switch will run all simulations and exit.
  • ApsimUI.exe is the conventional Windows User Interface.
  • ConToSim.exe, a console application, converts .con files to .sim files.
  • ApsimToSim.exe, a console application, converts .apsim files to .sim files. It requires the full path name of the .apsim file.

Windows Shell Extension (Explorer menu)

Right clicking on a .apsim, .con or .sim file in the windows explorer opens a menu to open the file.

 

Unix

Only apsim.x, ConToSim.exe and a shell equivalent of APSRun are used on unix.
