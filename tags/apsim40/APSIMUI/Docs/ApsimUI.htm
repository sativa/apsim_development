APSIM User Interface

 

Introduction

 

From version 4, APSIM comes with a new user interface that lets users configure simulations via a drag and drop centric interface. This interface, unlike APSFront, provides complete access to all APSIM parameters.

 

Getting started

 

When first started, the interface shows a menu and toolbar at the top, an empty simulation in the white box on the left and a help window at the bottom. The first step is to click New and select a simulation that is closest to the type of simulation you want to build.

 

 

Once loaded, a simulation is displayed in tree view form.

Toolboxes

Creating simulations from scratch, or modifying existing ones involves dragging modules from one or more toolboxes. By default, the interface comes with a standard toolbox that can be accessed by clicking on the toolbox button.

 

 

These modules can be dragged onto your simulation.

 

You can create your own toolbox by going to View | Options and clicking on New. This toolbox will then be accessible via the toolbox button and you can drag simulations or modules within a simulation onto your toolbox. This is a great way to reuse pieces of simulation, or complete simulations, in different projects.

 

Running APSIM

Running APSIM is as simple as clicking on the Run button on the toolbar. The resulting output file and summary files can be viewed by clicking on outputfile and summaryfile in the simulation tree.
