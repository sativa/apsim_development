APSVis user manual

 

Introduction

 

The Agricultural Production Systems sIMulator (APSIM) is a modelling framework that, when executed, can produce large amounts of output. This output can spread across many files, with each containing a diverse range of output variables. The APSVis application has been developed to provide a charting capability within the APSIM group of applications. APSVis can rapidly produce a range of charts from these APSIM model output files.

 

Starting APSVis

 

APSVis can be started in a variety of ways. The simplest is to select one or more output files in Windows Explorer, click the right mouse button and select Graph file. This will cause a switch to APSVis and automatic loading of the selected output files.

For APSFront users, select the View Output menu option, the Graph sub menu option, select one or more output files and then click the OK button. This will execute APSVis and automatically load the selected output files.

Regardless of which of these two methods is used to start APSVis, a form is then displayed that provides you with a range of chart types to choose from to display your data.
 

Predicted and Observed Files

 

APSVis has the capability of displaying both predicted and observed data on the same chart. The file format that APSVis reads for both output and observed files is as follows:

    Summary_file=test.sum
    Title =sorg / wheat
    year  day        date   iwh_cover_tot   iwh_grainn%
    ()     ()          ()           (0-1)           (%)
    1988    1   1/01/1988              0              0
    1988    2   2/01/1988              1              0

The summary_file= line is optional. Columns of data are separated by one or more spaces and each column must have a both a heading and a units line.

Observed files have an identical format to the predicted files. However, the summary_file= line is usually omitted and the title must match the title in the corresponding predicted file. You can denote missing data with a ‘-‘ symbol.

To indicate which predicted and observed files APSVis should use when drawing charts, either click, individually, on the first 2 buttons on the button bar or select from either of the Open predicted files… or Open observed files… items under the File menu. Select one or more files in the “Open” box. The files selected are used for all future charts until different files are selected by the user.

Printing and exporting charts

 


To print a chart, click within the chart you want to print (to make it the current chart) and then select the File|Print menu option.  This will display the following form:

 

This form allows you to select the printer you wish to use, the printer details, paper orientation etc.  One very useful option is the Chart Detail slider in the bottom left corner.  This changes the size of the chart and fonts from the display size and font to selections that are more appropriate for a printed page.

 To export a chart to the Windows clipboard so that it can be pasted into other applications, select the Edit|Copy menu item.  APSVis copies a Windows metafile representation of your chart to the clipboard.  You can then switch to another application that supports clipboard pasting and paste the chart into that application.

 

Zooming charts

APSVis contains a sophisticated zoom and scroll facility to allow the user to zoom the chart to any scale using the mouse and then scroll that chart.  To zoom press the left mouse button and drag a rectangular area downward. The area within the rectangle is zoomed to fill the window.  To scroll the zoomed graph, press the right mouse button on the chart and drag.

 

To reset a zoomed graph to full size, press the left mouse button and drag a rectangular area upward.


 


 

Scatter charts

You should select an X variable and 1 or more Y variables. In some situations you may want to treat the x variable in different ways. APSVis allows the x variable to be interpreted as a numeric, date or string.

If the data is treated as a string, APSVis will not evaluate the x data value but simply space all the values equally along the X-axis (Microsoft EXCEL refers to these types of charts as 'Line' charts). You also have the option of accumulating the X variable by selecting the Accumulate X box on the right.

For each Y variable you have the option of ticking a Y2 check box to indicate that that variable should be linked to the right-hand Y-axis. You also have the option of accumulating the Y variable by selecting the Accumulate YX box at the right of the individual Y variable. Every Y variable chosen has a chart-type drop down list that APSVis uses to create a chart series.


Depth charts

To create depth charts there are several requirements of the output file. There must be a column labelled Date in the file and a depth column. If using APSFront, ensure that Date (dd/mm/yyyy) is selected from the clock module, under the Reporting menu item along with Depth - layered (mm) from the Soil water module If not using APSFront, you should add the following lines to your report parameter file:

Variable = clock.dd/mm/yyyy as Date

Variable = soilwat2.dlayer

Frequency charts

 

 

 

One x variable can be selected along with a standard APSVis chart type (See Creating Scatter Charts). A frequency chart groups all the values into categories. The user can specify these categories.
