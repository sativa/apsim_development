Simulation Failure

The fatal error message.

Fatal errors are recorded in the summary file in a manner such as the following example.


!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!
! FATAL ERROR IN SYSTEM MODEL                     !     <------- Error Type
!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!
A fatal error was found. Routine traceback :-
Read_param
soilwat2_soil_profile_param                            <------- Routine Traceback
soilwat2_reset
soilwat2

Current module name :- soilwat2                        <------- Current Module
Error type :- User error                               <------- Fatal Error Type

Cannot find a parameter in any of the files/sections
specified in the control file. Parameter name = bd    <-------- Description
Section specifier = parameters
!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!

In the example we can see that a fatal error occurred in the Soilwat2 module's execution of the reading parameters for the soil profile.  The error occurred because of a failure to read the parameter 'bd' in the 'parameters' section and as such is a user error.
