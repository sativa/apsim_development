APSIM Fertiliz Module

Apsim Fertiliz


This routine is the interface between the main system and the fertiliz module.

Subprogram Logic



if (action.=.mes_presence) then
get_current_module module_name
Print 'module_name = ' , trim (module_name) / / blank / / version number
else if (action.=.mes_init) then
Initialise module state variables
Initialise the module state
else if (action.=.mes_inter_timestep) then
Reset daily state variables.
else if (action.=.mes_process) then
Get required state variables from other modules
Check the internal fertilizer schedule
else if (action.=.'apply'.or.action.=.'fertilize') then
Get required state variables from other modules
Apply the specified fertilizer
else if (action.=.mes_get_variable) then
Provide information to requesting module
else if (action.=.mes_set_variable) then
Set an internal variable as requested
else
message_unused
endif

Fertiliz Apply


apply fertiliser as directed

Subprogram Logic


if (amount.>.0.0) then
Write a string to a file
layer = Find index for cumulative dlayer = depth
Read full_name from file
Read components from file
Read fraction from file (Lower Bound = number of values returned, Upper Bound = 0 . 0)
For each counter from 1 to number of values returned
Get array from the system (Lower Bound = 0 . 0, Upper Bound = 1 . 0e30)(OPTIONAL)
if (array_size.>.0) then
get_posting_module owner_module
Fill delta_array with 0 . 0
delta_array layer = amount * fraction counter

Dname = 'dlt_' / / components counter
Supply delta_array to the messaging system using the name Dname
message_send_immediate unknown_module, mes_set_variable, Dname

else
endif
next
fert_applied = fert_applied + amount
write (to string) amount , ' of ' , trim (full_name) , ' , trim (type) , ')' , new_line , 'added at depth ' , depth , ' (layer) , layer , ')'
Report this event to the system
else
endif

Fertiliz Fertilize


This routine deciphers the data string and applies the fertiliser.

Subprogram Logic


Get amount from the message (Lower Bound = 0 . 0, Upper Bound = 1000 . 0)
Get depth from the message (Lower Bound = 0 . 0, Upper Bound = 1000 . 0)(OPTIONAL)
if (number of values returned_depth.=.0) then
depth = 0 . 0
else
endif
Get type from the message
if ((number of values returned_type.=.0).or.(number of values returned_amount.=.0)) then
Report a fatal error
else
Pass the fertilizer information to other modules
endif
