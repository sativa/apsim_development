Simulation Failure

The Unhandled Exception.

In this case, the program code was unable to deal with an error in execution.  In order to resolve this issue we must determine the nature of the initial error, or more importantly, the cause of the initial error.

  1. Make a note of the error message observed on the screen. 
    This may have been text in the simulation window or on a dialogue box.  Include all text including any routine trace-back tables etc for use in documenting the problem as well as your personal problem resolution.

  2. Check the simulation summary file for Fatal Error Messages.
    If the summary file does include Fatal Errors then treat the problem as a handled exception.
    Go to the Help for Handled Exceptions.

  3. Check the summary file for warning messages
    for the period of the simulation just prior to the failure.  Document any that do occur.

  4. Unable to Solve the Problem?
    If you are unable to find a sensible reason for the failure then contact the APSIM Help Desk for assistance.

    Go to Asking the Help Desk for Help.
