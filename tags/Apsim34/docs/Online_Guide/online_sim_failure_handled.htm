Simulation Failure

The Handled Exception (Fatal error)
In the case of a handled exception, a Fatal Error Message will be recorded in the simulation summary file.

It is always advisable to only look into the first recorded fatal error.  The contents of the error message will help you determine the source of your problem.   Common problems relate to incomplete input data, bad module communications or syntactical errors in parameter files.

Read the error message carefully, noting the name of the module registering the error and the description of the error.  In most cases the information provided will be adequate to solve the problem.
