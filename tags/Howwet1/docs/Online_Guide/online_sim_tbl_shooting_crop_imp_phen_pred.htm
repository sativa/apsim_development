Crop growth & development

Improving phenological predictions

In improving phenological predictions it is important to fit the model to observations of early crop development before considering later crop development.  This is a simple point but it is often lost during model application.  There are a few things to check before turning to phenological parameters as a means of simulation improvement.

Things to check

  1. Do you have the correct sowing date in the simulation? (Check the summary file - Not the parameter file)

  2. Do you have the correct sowing depth in the simulation? (Check the summary file - Not the parameter file)

  3. Do you have the correct cultivar in the simulation? (Check the summary file - Not the parameter file)

  4. Do you have adequate confidence in the observed crop development measurements?

  5. Is the weather data (especially temperature) adequate for the site you are dealing with?

If any of the above needed adjustment rerun the simulation and start again.

The phenological development of crops can be difficult to simulate, especially for a range of cultivars grown.  Contact the module support staff for help in choosing adequate cultivar parameters.
